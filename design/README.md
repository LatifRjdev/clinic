# Design Prompts — Clinic MIS

Промпты для генерации дизайна всех экранов системы управления клиникой (формат Stitch).

## Общие параметры дизайна

- **Стиль:** Чистый, медицинский, профессиональный
- **Цвета:** #1E40AF (синий), #F8FAFC (фон), #10B981 (зелёный), #EF4444 (красный), #F59E0B (жёлтый)
- **Шрифт:** Inter — 24px bold заголовки, 16px semibold подзаголовки, 14px regular текст
- **Иконки:** Lucide / Phosphor — минималистичные линейные
- **Платформа:** Web, desktop-first (mobile-first для пациентов)
- **Языки:** Русский (основной), Узбекский (переключение)
- **Layout:** Sidebar 240px→64px collapsible + Header 56px sticky
- **Компоненты:** Sidebar, хедер с поиском/уведомлениями, карточки, таблицы, модалки, тосты

## Формат промптов (Stitch)

Каждый промпт содержит 7 секций:
1. **Визуальный стиль** — цвета, типографика, пространство
2. **Макет и структура** — layout, grid, колонки, размеры
3. **Элементы UI** — компоненты, размеры, состояния
4. **UX-аспекты** — взаимодействия, hover, анимации, transitions
5. **Брендинг** — логотип, визуальный язык
6. **Доступность** — ARIA, контраст, фокус, клавиатура
7. **Тренды** — glassmorphism, neumorphism, градиенты, skeleton loading
