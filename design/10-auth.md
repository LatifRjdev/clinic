# Авторизация и онбординг — Design Prompts (Stitch)

> Экраны входа, двухфакторной аутентификации, восстановления пароля, первоначальной настройки клиники и приглашения сотрудников.

---

## 10.1 Страница входа (Login)

```
Разработайте современный, профессионально выглядящий интерфейс страницы входа в медицинскую информационную систему клиники.

Визуальный стиль:
- Двухколоночный layout, внушающий доверие и профессионализм медицинского учреждения
- Цветовая палитра: основной #1E40AF (глубокий синий), фон #F8FAFC (светло-серый), акцент #10B981 (зелёный), ошибки #EF4444 (красный)
- Левая колонка — насыщенный градиент от #1E40AF к #3B82F6 с анимированным волнообразным паттерном (SVG wave), поверх — полупрозрачные геометрические медицинские элементы (пульс-линия, крест, стетоскоп в line-art стиле)
- Правая колонка — чистый белый фон (#FFFFFF) с лёгкой тенью карточки формы (box-shadow: 0 4px 24px rgba(0,0,0,0.08))
- Типографика Inter: заголовок формы 28px/700 (bold), подзаголовки 16px/500, лейблы 14px/500, body 14px/400
- Общее ощущение: чистота, надёжность, современная медицина

Макет и структура:
- Левая часть (60% ширины): анимированный градиентный фон #1E40AF → #3B82F6 с плавным wave-эффектом (CSS animation, бесконечный цикл, медленное движение волн)
  - В центре — hero-иллюстрация медицинской тематики (доктор с планшетом, или интерьер клиники) в flat/line-art стиле, белые и полупрозрачные тона
  - Под иллюстрацией — название клиники крупным белым шрифтом (32px/700), например "MedClinic Pro"
  - Слоган курсивом или light-весом: "Надёжная система для вашей клиники" (18px/300, белый с opacity 0.85)
  - Декоративные элементы: полупрозрачные круги, точечный паттерн, пульс-линия — всё анимировано (subtle float animation)
- Правая часть (40% ширины): белый фон, вертикально и горизонтально отцентрированная карточка формы (max-width 380px, padding 40px)
  - Логотип клиники сверху (высота 48px, отцентрирован)
  - Отступ 24px
  - Заголовок "Вход в систему" (28px/700, цвет #1E293B)
  - Отступ 32px
  - Поле "Логин или email" — Input height 48px, border-radius 10px, border 1.5px #E2E8F0, padding-left 44px (место для иконки User из Lucide, 20px, цвет #94A3B8), placeholder "Введите логин или email", focus: border-color #1E40AF + ring 3px rgba(30,64,175,0.15)
  - Отступ 16px
  - Поле "Пароль" — аналогичный стиль, иконка Lock слева, справа — кнопка-toggle Eye/EyeOff (Lucide), при клике переключает type password/text, иконка 20px, цвет #94A3B8, hover #64748B
  - Отступ 16px
  - Строка между: слева — чекбокс "Запомнить меня" (custom checkbox 18x18, border-radius 4px, checked: bg #1E40AF с белой галочкой, label 14px/400 #64748B), справа — ссылка "Забыли пароль?" (14px/500, цвет #1E40AF, hover: underline)
  - Отступ 24px
  - Кнопка "Войти" — full-width, height 48px, bg #1E40AF, hover bg #1E3A8A, border-radius 10px, текст 16px/600 белый, transition 200ms, cursor pointer
  - Отступ 32px
  - Разделительная линия (1px #E2E8F0)
  - Отступ 16px
  - Переключатель языка внизу: сегментированный контрол (RU | UZ), width 160px по центру, active сегмент bg #EFF6FF текст #1E40AF, inactive bg transparent текст #94A3B8, border-radius 8px, border 1px #E2E8F0

Элементы UI:
- Input fields: height 48px, border-radius 10px, border 1.5px solid #E2E8F0, bg #FFFFFF, icon prefix 20px (Lucide: User, Lock)
- Password toggle: кнопка Eye/EyeOff (Lucide/Phosphor), расположена внутри input справа, padding-right 44px
- Primary button: full-width, bg gradient subtle от #1E40AF к #2563EB, border-radius 10px, shadow 0 2px 8px rgba(30,64,175,0.25)
- Checkbox: custom, 18x18px, border-radius 4px, transition 150ms
- Language switcher: segmented control, animated slide indicator
- Hero background: CSS animated gradient + SVG wave overlay + floating medical illustrations

UX-аспекты:
- Autofocus на поле логина при загрузке страницы
- Enter в любом поле — submit формы
- Состояние ошибки: вся карточка формы выполняет shake-анимацию (CSS keyframes, 300ms, горизонтальное покачивание), поля input получают border-color #EF4444, под формой появляется сообщение "Неверный логин или пароль" (14px, цвет #EF4444, иконка AlertCircle из Lucide слева, bg #FEF2F2, padding 12px, border-radius 8px)
- Состояние загрузки: текст кнопки заменяется на спиннер (border-spinner 20px, белый, animation rotate 600ms) + кнопка disabled с opacity 0.7
- "Запомнить меня" сохраняет сессию на 30 дней, иначе — до закрытия браузера
- Tab-навигация: логин → пароль → запомнить → войти → забыли пароль → язык
- Валидация: пустые поля — красная рамка + "Обязательное поле" под input

Брендинг:
- Логотип клиники в правой части (над формой) и крупное название на левой hero-части
- Цвет #1E40AF пронизывает весь интерфейс: кнопки, фокусы, ссылки, hero-фон
- Слоган отражает надёжность и профессионализм
- Иконография медицинской тематики на hero-части подчёркивает отрасль
- Общий tone-of-voice: спокойный, уверенный, профессиональный

Доступность:
- Все input имеют видимые label (расположены над полем или как floating label)
- Ошибки анонсируются через aria-live="assertive"
- Контрастность текста на фоне соответствует WCAG AA (минимум 4.5:1)
- Форма обёрнута в <form> с role, кнопка type="submit"
- Password toggle имеет aria-label "Показать пароль" / "Скрыть пароль"
- Focus ring visible (3px ring #1E40AF с opacity)
- Skip-to-content link для screen readers

Тренды:
- Анимированный градиентный фон на hero-части с плавным wave-движением (CSS @keyframes)
- Glassmorphism-эффект на карточке формы: backdrop-filter blur(12px), bg rgba(255,255,255,0.85), border 1px rgba(255,255,255,0.5) — если поверх цветного фона
- Input focus ring с мягким glow-эффектом (box-shadow expand animation)
- Кнопка "Войти" с micro-interaction: при нажатии scale(0.97) 100ms, при hover — subtle gradient shift
- Floating medical icons на hero-части с parallax-эффектом (subtle, не отвлекающий)
- Smooth entrance animation: форма появляется с fadeIn + translateY(20px) → 0, 400ms ease-out
```

## 10.2 Двухфакторная аутентификация (2FA)

```
Разработайте современный, профессионально выглядящий интерфейс экрана двухфакторной аутентификации для медицинской информационной системы клиники.

Визуальный стиль:
- Минималистичный, security-oriented дизайн, фокус на единственном действии — ввод кода
- Цветовая палитра: основной #1E40AF (синий), фон — мягкий градиент от #F8FAFC к #EFF6FF (очень лёгкий голубоватый), карточка #FFFFFF, ошибки #EF4444, успех #10B981
- Типографика Inter: заголовок 24px/700, описание 15px/400 цвет #64748B, code digits 28px/700
- Ощущение: безопасность, простота, быстрота действия
- Крупные, хорошо читаемые поля для кода — визуальный центр экрана

Макет и структура:
- Фон: мягкий градиент #F8FAFC → #EFF6FF, по центру — белая карточка (max-width 440px, padding 48px 40px, border-radius 16px, shadow 0 8px 32px rgba(0,0,0,0.08))
- Структура карточки сверху вниз:
  - Логотип клиники (height 40px, по центру)
  - Отступ 24px
  - Иконка Shield (Lucide) в круге: bg #EFF6FF, border 2px #1E40AF, size 64px, иконка 28px цвет #1E40AF
  - Отступ 20px
  - Заголовок "Двухфакторная аутентификация" (24px/700, #1E293B, text-align center)
  - Отступ 8px
  - Описание "Код подтверждения отправлен на +998 ** *** 45" (15px/400, #64748B, text-align center)
  - Отступ 32px
  - Табы переключения метода (если несколько): сегментированный контрол (SMS | Email | Приложение), стиль как language switcher, active bg #EFF6FF text #1E40AF, border-radius 8px
  - Отступ 24px
  - 6 отдельных input-полей для цифр: каждое 52x60px, border-radius 12px, border 2px #E2E8F0, bg #FFFFFF, font 28px/700 #1E293B, text-align center, gap между полями 12px, сгруппированы по центру
  - Отступ 20px
  - Таймер повторной отправки: "Отправить повторно через 0:59" (14px/400, #94A3B8, text-align center), когда таймер истёк — ссылка "Отправить код повторно" (14px/500, #1E40AF, hover underline)
  - Отступ 24px
  - Кнопка "Подтвердить" — full-width, height 48px, bg #1E40AF, text 16px/600 белый, border-radius 10px, disabled пока не все 6 цифр заполнены (opacity 0.5)
  - Отступ 16px
  - Ссылка "← Вернуться к входу" (14px/500, #64748B, hover #1E40AF, text-align center)
- Вариант Authenticator App: вместо OTP-полей — QR-код (200x200px, рамка border 1px #E2E8F0, border-radius 12px, padding 16px) + под ним "Или введите код вручную:" + monospace текст секретного ключа (14px, bg #F1F5F9, padding 12px, border-radius 8px, copy-button рядом)

Элементы UI:
- OTP input поля: 6 отдельных input[maxlength=1], size 52x60px, font 28px/700, border-radius 12px
- Авто-фокус: при загрузке фокус на первом поле, при вводе цифры — авто-переход на следующее
- Paste поддержка: при вставке 6-значного кода — автозаполнение всех полей
- Авто-submit: при заполнении 6-го поля автоматическая отправка
- Backspace: удаление цифры и возврат фокуса на предыдущее поле
- Таймер обратного отсчёта: 60 секунд, обновление каждую секунду, анимированный (smooth countdown)
- Tab switcher для методов: SMS / Email / Authenticator (Lucide иконки: Smartphone, Mail, Key)

UX-аспекты:
- Auto-focus на первом OTP-поле при открытии экрана
- Ввод только цифр (inputmode="numeric", pattern="[0-9]")
- При вставке из буфера — автозаполнение всех 6 полей и авто-submit
- При ошибке: все 6 полей получают border-color #EF4444, shake-анимация (300ms), текст ошибки "Неверный код. Осталось N попыток" (14px, #EF4444, иконка AlertCircle), после анимации поля очищаются и фокус на первое
- При успехе: все поля получают border-color #10B981, иконка CheckCircle в центре, 500ms пауза → redirect
- Resend доступен только после истечения таймера (60 секунд)
- После 3 неудачных попыток: блокировка на 5 минут с таймером
- Переключение метода 2FA через табы без перезагрузки страницы

Брендинг:
- Логотип клиники сверху карточки обеспечивает узнаваемость
- Иконка Shield в фирменном синем цвете #1E40AF подчёркивает безопасность
- Цветовая палитра консистентна с login-страницей
- Заголовок на русском языке, профессиональная медицинская терминология
- Общий тон: "Мы защищаем ваши данные"

Доступность:
- Каждое OTP-поле имеет aria-label "Цифра 1 из 6", "Цифра 2 из 6" и т.д.
- Таймер обратного отсчёта обновляется через aria-live="polite"
- Сообщения об ошибках через aria-live="assertive"
- Focus ring на OTP-полях (3px ring #1E40AF)
- Вся навигация доступна через Tab/Shift+Tab
- Screen reader анонсирует переключение метода 2FA
- Кнопка "Подтвердить" имеет aria-disabled когда не все цифры введены

Тренды:
- OTP-поля с glassmorphism-эффектом при фокусе: bg rgba(239,246,255,0.6), backdrop-filter blur(8px)
- Shake-анимация при ошибке (CSS keyframes: translateX ±8px, 300ms)
- Success-анимация: SVG draw check-mark (stroke-dasharray анимация 500ms)
- Плавный countdown-таймер с CSS transition
- Карточка появляется с fadeIn + scale(0.95→1) анимацией, 300ms ease-out
- Мягкий pulse-эффект на иконке Shield при загрузке
- Subtle animated gradient на фоне (медленное движение, 15s цикл)
```

## 10.3 Восстановление пароля — Шаг 1: Запрос

```
Разработайте современный, профессионально выглядящий интерфейс первого шага восстановления пароля (запрос кода) для медицинской информационной системы клиники.

Визуальный стиль:
- Чистый, минималистичный wizard-дизайн с чётким прогрессом по шагам
- Цветовая палитра: основной #1E40AF, фон #F8FAFC, карточка #FFFFFF, прогресс-бар активный #1E40AF, неактивный #E2E8F0
- Типографика Inter: заголовок 24px/700, описание 15px/400 #64748B, label 14px/500
- Атмосфера: поддержка пользователя, понятный путь, нет тревоги
- Индикатор шагов явно показывает "вы на шаге 1 из 4"

Макет и структура:
- Фон: сплошной #F8FAFC, по центру экрана (вертикально и горизонтально) — белая карточка (max-width 440px, padding 48px 40px, border-radius 16px, shadow 0 8px 32px rgba(0,0,0,0.08))
- Индикатор шагов вверху карточки: 4 круга (32px) с номерами (1, 2, 3, 4), соединённых линиями
  - Шаг 1: круг bg #1E40AF, текст белый, линия до шага 2 — #E2E8F0
  - Шаги 2-4: круг border 2px #E2E8F0, текст #94A3B8, линии #E2E8F0
  - Подписи под кругами: "Запрос" | "Код" | "Пароль" | "Готово" (12px/500, #94A3B8, active #1E40AF)
- Отступ 32px
- Иконка Mail (Lucide) в круге: bg #EFF6FF, size 56px, иконка 24px #1E40AF
- Отступ 16px
- Заголовок "Восстановление пароля" (24px/700, #1E293B, text-align center)
- Описание "Введите email или номер телефона, привязанный к вашему аккаунту. Мы отправим код подтверждения." (15px/400, #64748B, text-align center, max-width 340px, margin auto)
- Отступ 28px
- Input "Email или телефон" — height 48px, border-radius 10px, border 1.5px #E2E8F0, иконка AtSign (Lucide) слева 20px #94A3B8, placeholder "example@clinic.uz или +998...", focus border #1E40AF + ring
- Отступ 24px
- Кнопка "Отправить код" — full-width, height 48px, bg #1E40AF, text 16px/600 белый, border-radius 10px
- Отступ 16px
- Ссылка "← Вернуться к входу" (14px/500, #64748B, hover #1E40AF, text-align center, с иконкой ArrowLeft 16px)

Элементы UI:
- Step indicator: 4 numbered circles (32px) + connecting lines (2px height), animated fill
- Input с иконкой: height 48px, icon prefix (AtSign или Phone auto-detect при вводе)
- Primary button: full-width, bg #1E40AF, hover #1E3A8A, border-radius 10px
- Back link: стрелка + текст, subtle style
- Карточка: white, rounded 16px, shadow medium

UX-аспекты:
- Auto-detect типа ввода: если начинается с "+" или цифры — переключить иконку на Phone, иначе AtSign (Mail)
- Валидация: проверка формата email (regex) или телефона (+998XXXXXXXXX) перед отправкой
- Ошибка "Аккаунт не найден": красный текст под input, border #EF4444, иконка AlertCircle
- Ошибка "Слишком много попыток": блок с предупреждением (bg #FEF3C7, border-left 4px #F59E0B, иконка AlertTriangle, текст "Попробуйте через 5 минут")
- Loading state на кнопке: спиннер, disabled
- При успехе: плавный переход (slide-left) на шаг 2
- Autofocus на input при загрузке

Брендинг:
- Логотип клиники опционально наверху (если есть место) или в step indicator area
- Цвет #1E40AF как основной акцент: шаг, кнопка, фокус, ссылки
- Текст на русском, понятные формулировки без технического жаргона
- Тон: "Мы поможем вам восстановить доступ"
- Иконография из набора Lucide: Mail, AtSign, ArrowLeft, AlertCircle

Доступность:
- Input с видимым label "Email или телефон" (можно floating label, но visible в обоих состояниях)
- Step indicator: aria-label="Шаг 1 из 4: Запрос кода", aria-current="step"
- Ошибки: aria-live="assertive", связаны с input через aria-describedby
- Focus management: при загрузке фокус на input
- Кнопка: aria-busy="true" в loading state
- Навигация клавишами: Tab проходит input → кнопка → ссылка "вернуться"

Тренды:
- Step indicator с animated connecting lines: при переходе на шаг — линия заполняется цветом слева направо (CSS transition width, 400ms)
- Карточка с subtle entrance animation: fadeIn + translateY(16px→0), 350ms ease-out
- Input focus: border-color transition 200ms + box-shadow expand (ring glow)
- Кнопка hover: gradient shift + shadow increase, 200ms transition
- Micro-animation на иконке Mail: лёгкий bounce при загрузке страницы
- Auto-detect input type с плавной сменой иконки (crossfade 200ms)
```

## 10.4 Восстановление пароля — Шаг 2: Ввод кода

```
Разработайте современный, профессионально выглядящий интерфейс второго шага восстановления пароля (ввод кода подтверждения) для медицинской информационной системы клиники.

Визуальный стиль:
- Тот же минималистичный wizard-стиль, что и шаг 1, с обновлённым прогресс-индикатором
- Цветовая палитра: #1E40AF (основной), #F8FAFC (фон), #FFFFFF (карточка), #10B981 (завершённый шаг), #EF4444 (ошибка)
- Типографика Inter: заголовок 24px/700, описание 15px/400 #64748B, OTP цифры 28px/700
- Визуальная преемственность с шагом 1 — тот же размер карточки, отступы, стиль

Макет и структура:
- Фон #F8FAFC, центрированная белая карточка (max-width 440px, padding 48px 40px, border-radius 16px, shadow)
- Step indicator вверху: шаг 1 — зелёный круг с галочкой (bg #10B981, иконка Check белая), линия 1→2 заполнена #1E40AF, шаг 2 — активный (bg #1E40AF, текст "2" белый), шаги 3-4 неактивны
- Отступ 32px
- Иконка MessageSquare (Lucide) в круге: bg #EFF6FF, 56px, иконка 24px #1E40AF
- Отступ 16px
- Заголовок "Введите код подтверждения" (24px/700, #1E293B, center)
- Описание "Мы отправили 6-значный код на email@example.com" (15px/400, #64748B, center) — адрес или телефон жирным (#1E293B)
- Отступ 28px
- 6 OTP-полей: каждое 52x60px, border-radius 12px, border 2px #E2E8F0, font 28px/700 #1E293B, text-align center, gap 12px, сгруппированы по центру
- Отступ 20px
- Таймер: "Отправить повторно через 0:59" (14px/400, #94A3B8, center), по истечении — ссылка "Отправить код повторно" (14px/500, #1E40AF)
- Отступ 24px
- Кнопка "Подтвердить" — full-width, height 48px, bg #1E40AF, disabled до заполнения всех 6 цифр
- Отступ 16px
- Ссылка "← Назад" (14px/500, #64748B)

- Focus management: при переходе на шаг — фокус на заголовок шага
- Confetti: decorative, aria-hidden="true", respect prefers-reduced-motion

- Glassmorphism на финальной карточке поверх confetti
- Micro-interactions: кнопки "+ Добавить" с scale bounce при hover

## 10.8 Первый вход приглашённого сотрудника (Employee Invitation)

```
Разработайте современный, профессионально выглядящий интерфейс первого входа приглашённого сотрудника в медицинскую информационную систему клиники — экран регистрации по приглашению и последующий guided tour.
