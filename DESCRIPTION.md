# Clinic MIS — Описание системы

Медицинская информационная система (МИС) для управления многопрофильной клиникой. Единая веб-платформа, объединяющая все процессы клиники — от записи пациента до финансовой отчётности.

---

## Роли системы

| Роль | Что делает |
|------|-----------|
| **Владелец клиники** | Управляет бизнесом: финансы, персонал, филиалы, аналитика, прайс-лист, маркетинг |
| **Главный врач** | Контролирует медицинскую часть: расписание врачей, протоколы лечения, качество, статистика |
| **Врач** | Ведёт приём: осмотр, диагноз, назначения, рецепты, направления, телемедицина |
| **Медсестра** | Выполняет процедуры, вводит витальные показатели, ведёт журнал, управляет расходниками |
| **Рецепшен** | Записывает пациентов, регистрирует, управляет очередью, принимает оплату, печатает документы |
| **Бухгалтер** | Ведёт доходы/расходы, зарплаты, ДМС-расчёты, налоги, работает с контрагентами и отчётами |
| **Сисадмин** | Управляет пользователями, ролями, интеграциями, бэкапами, логами, настройками системы |
| **Пациент** | Записывается онлайн, просматривает результаты, общается с врачом, оставляет отзывы |

---

## Функции системы

### Пациенты
- Регистрация и ведение базы пациентов (ФИО, контакты, паспорт, ДМС-полис, аллергии)
- Поиск по ФИО, телефону, ID, паспорту
- Карточка пациента с полной историей: визиты, ЭМК, документы, счета
- Теги, группы, объединение дублей

### Расписание и запись
- График работы врачей (дни, часы, перерывы, длительность слотов)
- Запись на приём (рецепшен или онлайн пациентом)
- Визуальное расписание: день / неделя / месяц
- Управление кабинетами и проверка конфликтов
- Живая очередь в клинике
- Автоматические SMS-напоминания (за 24ч и 2ч)
- Подтверждение, отмена, перенос записи

### Электронная медицинская карта (ЭМК)
- Ведение приёма: жалобы → анамнез → осмотр → диагноз (МКБ-10) → рекомендации
- Шаблоны осмотров по специальностям
- Назначения и рецепты (медикаменты, дозировки, курс)
- Направления к другим специалистам
- Электронная подпись врача (черновик → подписана)
- Витальные показатели (давление, пульс, температура, вес, рост)
- Прикрепление файлов (снимки, результаты анализов)

### Биллинг и касса
- Каталог услуг с ценами и категориями
- Автоматическое формирование счетов после приёма
- Оплата: наличные, карта, перевод, ДМС
- Скидки (процент / фиксированная сумма)
- Касса: открытие/закрытие смены, инкассация
- Печать чеков
- Возвраты с подтверждением

### Финансы
- Доходы и расходы по категориям
- Зарплаты: оклад + процент от услуг, ведомости, утверждение, выплата
- ДМС-расчёты: реестры услуг, акты сверки со страховыми
- Налоговый календарь и калькулятор
- Работа с контрагентами (поставщики, арендодатели)
- Финансовые отчёты: P&L, Cash Flow, по услугам, по врачам, конструктор

### Склад
- Учёт медикаментов и расходных материалов
- Приход, расход, списание
- Контроль минимальных остатков и сроков годности
- Привязка расхода к процедуре/приёму

### Коммуникации
- Внутренний чат между сотрудниками (1-на-1 и группы)
- Чат врач ↔ пациент
- Обмен файлами, сообщения в реальном времени

### Уведомления
- In-app push-уведомления (реальное время)
- SMS-уведомления (напоминания, коды подтверждения)
- Email-уведомления (отчёты, приглашения)
- Telegram-бот (запись, напоминания)
- Настройка каналов по типу события

### Телемедицина
- Видеоконсультации (WebRTC)
- Чат во время сеанса
- Предварительная анкета пациента
- Оценка после консультации

### Задачи
- Создание задач, назначение исполнителя
- Приоритеты и дедлайны
- Статусы: новая → в работе → на проверке → завершена
- Комментарии к задачам

### Документы
- Загрузка и хранение файлов (MinIO / S3)
- Шаблоны документов (справки, направления, заключения)
- Генерация PDF из шаблона + данные пациента
- Печать

### Аналитика
- Дашборды по ролям с KPI-карточками и графиками
- Статистика приёмов, выручки, загрузки врачей
- Маркетинг: источники пациентов, конверсия
- Удовлетворённость пациентов (оценки, NPS)
- Экспорт отчётов (PDF, Excel)

### Маркетинг
- Отслеживание источников пациентов
- SMS/Email рассылки
- Акции и скидки
- Программа лояльности
- Отзывы и рейтинг

### Администрирование
- Управление пользователями (создание, блокировка, сброс пароля)
- Гибкая настройка ролей и прав доступа (матрица)
- Интеграции: SMS-шлюз, платёжные системы, ЛИС, ЕГИСЗ, SMTP, Telegram
- Резервное копирование (автоматическое + ручное, восстановление)
- Системные логи (structured + raw, real-time)
- Мониторинг (CPU, RAM, диск, статус сервисов)
- Обновления системы

### Безопасность
- JWT-авторизация (access + refresh токены)
- Двухфакторная аутентификация (SMS, Email, Authenticator)
- Блокировка после неудачных попыток входа
- Журнал аудита всех действий
- Soft delete (данные не удаляются физически)
- IP whitelist (опционально)

### Мультифилиальность
- Поддержка нескольких филиалов
- Выбор филиала при входе
- Данные фильтруются по филиалу
- Пациенты — глобальные (доступны во всех филиалах)

### Локализация
- Русский (основной) и Узбекский
- Переключение языка в интерфейсе
- Форматы: дата DD.MM.YYYY, валюта UZS, телефон +998
